# Online Pet Shop (Frontend)

Frontend project of an online pet store built with **React** and **Redux Toolkit**.  

This is a **frontend-only project** that interacts with a provided backend API (no backend code was written by me). The project demonstrates modern React skills, reusable components, state management, routing, and responsive design.

---

## Features

- Browse and filter products by category
- Add or remove products from the shopping cart
- Product detail view
- Routing with React Router
- Breadcrumbs for easy navigation
- Reusable React components and utility functions
- Responsive and adaptive design for desktop and mobile

---

## Technologies

- **Frontend:** React, Redux Toolkit, JavaScript (ES6+), CSS  
- **Routing & State:** React Router, Redux Toolkit  
- **Build Tool:** Vite (development and build tool)  
- **API Requests:** Axios / Fetch  
- **Other:** Responsive layout, reusable components
